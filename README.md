Welkom op mijn GitHub project!

Dit project startte met een opdracht voor informatica; bouw je eigen website. Ik had geen idee waar ik het over wilde doen, totdat een vriend van mij het idee gaf om voor de KKToe een website te maken. De KKToe is bij ons op school de feestcommissie waar ik zelf ook altijd mee help. Zelf verzorg ik met een leuke club mensen de techniek, als hoofd-geluid. Ook werk ik bij Gigant in Apeldoorn waar we een soort website hebben met een rooster, waar je je kan inschrijven voor diensten, je diensten kan zien etc. Ik dacht dat dit ook wel heel handig zou zijn voor bij ons bij de KKToe. Niet alleen voor de techniek, maar ook voor de reguliere KKToe crew die zich bezig houden met versieren van de school etc. Inmiddels is dit project flink uit de hand gelopen en wordt het steeds serieuzer. Toen ik het liet zien aan mijn informatica docent zei hij direct al dat het er echt heel goed uit zag! Hij was flink verbaasd en zei tegen mij dat als ik het wil, ik van heb een domeinnaam krijg zodat we hem echt kunnen gaan gebruiken!
